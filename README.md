# 🧮 Simple Calculator Application – Manual Testing Project

## 📌 Project Overview
This project contains detailed manual test cases for a Simple Calculator Application that performs basic arithmetic operations.

The objective is to validate the correctness, reliability, and error handling of the calculator when performing mathematical operations.

---

## 🎯 Scope of Testing

The calculator supports the following operations:

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (/)

Testing includes both valid and invalid scenarios to ensure functional accuracy and proper error handling.

---

## 🧪 Test Coverage

### ✅ Valid Input Scenarios
- Addition of positive numbers
- Addition with negative numbers
- Subtraction operations
- Multiplication of integers and decimals
- Division resulting in integer values
- Division resulting in decimal values
- BODMAS rule validation (operator precedence)
- Complex mathematical expressions

### ❌ Invalid Input Scenarios
- Division by zero
- Non-numeric character input
- Multiple operators in sequence
- Empty input
- Incomplete expressions
- Invalid decimal formats

## 📌 Assumptions

- The calculator evaluates expressions based on BODMAS rules.
- Error messages are displayed for invalid inputs.
- The application should not crash under any invalid scenario.
